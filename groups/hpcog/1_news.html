对外开放单机模拟器SandUPSim，相关情况请参阅项目网页
[2004-11-26]

为配合计算所开放日活动，小组决定对外开放单机模拟器――兼容Linux应用环境的多粒度全系统模拟软件，SandUPSim 1.0版。该系统已经取得了国家计算机软件著作权登记证书。目前软件代码及文档正在整理中，将在近期内发布。
[2004-10-25]

高文老师从日本归国回到我们小组，她长期以来一直从事可靠性相关的研究工作，以后会和大家一起工作。所有组员对高文老师的到来表示热烈的欢迎。
[2004-10-17]

张立新博士(IBM Austin Research Lab)来到计算所进行学术交流，以《The Simulator for PERCS and Bluegene, and researches on memory controller support for applications》为题进行了学术报告。报告分为三个部分，分别讨论《The Impulse Adaptable Memory System》、 《Active Memory Operations》、 《PercsSim-A Full System Simulatior for PERCS》。
