项目简介

       Memory Box将分布在主机端的存储部件分离出来，以rack或server的形式，为多个节点提供远程内存服务。内存模块可以根据需要动态分配和释放，与主机动态绑定，从而实现高度共享，降低空闲率，减少资源浪费；同时，由于资源集中管理，便于故障后的快速替换恢复和管理；另外，memory box的出现，还可以为某些应用模式的改进提供了条件。Memory Box的核心包括三部分：远程内存服务器（Remote Memory Box，RMB），支持RMB访问的通信网络和协议和支持RMB访问的主机接口。
