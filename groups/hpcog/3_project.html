光互联的特点是支持数据长距离传输和高传输带宽,这就为更大空间尺度上计算部件之间的高速数据交换和共享(网格化)提供了可能。这种特性很可能会为计算机体系结构带来深刻的变革 ....

 Network/Grid RAM

Tron

      网络内存（Network RAM） 是相对于计算机自身的物理内存而言的。一台计算机除了可以访问自身的物理内存，也可以通过网络连接访问远端计算机的物理内存，这种远端内存就看作是网络内存 ....

   TRON 是目前世界上运用最广泛的嵌入式操作系统，已广泛应用在汽车电子、手机、数码相机和其它电子产品。在家具、工业产品设计、建筑、城市、博物馆、购物超市、物流网等领域已大量采用基于TRON 标准的嵌入式网络计算机 ....

Memory Box
 SandFox是基于网格化动态自组织体系结构（DSAG）的全系统并行模拟器，可以模拟SMP、MPP、Scalable MP等体系结构, 其目标是模拟支持体系结构按需定制(Architecture On Demand)的新型体系结构 ....

Memory Box将分布在主机端的存储部件分离出来，以rack或server的形式，为多个节点提供远程内存服务。内存模块可以根据需要动态分配和释放...
