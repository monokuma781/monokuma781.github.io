INTRANET软件结构及信息安全控制研究


现今，Internet/Intranet的发展正方兴未艾，Internet在全世界已经得到了广泛的应用。而Intranet作为Internet的关键技术如TCP/IP、HTTP以及WWW等应用到公司内部也得到了迅速的发展。但是现在Intranet系统在企业的应用还面临很多问题，比如还没有有效的信息组织方式将企业存在的各种各样的数据信息包括结构化和非结构化的数据统一的放到Intranet系统上进行管理，同时对数据信息的访问控制管理不完善，往往由于安全原因不能将一些重要的机密信息放到系统中去。

元数据的概念为数据表示提供了抽象模型，通过借鉴面向对象的思想，将企业内部的各种信息组织成对象的形式，用属性的方式定义数据对象，将属性分成基本属性和扩展属性，同时针对基本属性和扩展属性进行信息的访问控制。

利用元数据模型组织和管理数据为企业信息的组织和表示都带来了极大的方便性和灵活性，企业在建立Intranet系统时只需将信息分类并进行简单的配置定义。这种数据的表示使大量数据的细粒度的授权和安全控制成为现实。基于元数据的信息访问控制方法为Intranet系统提供了新的访问控制模式。

关键词：企业网络、数据表示、安全控制、对象技术、软件模式

Today，although the development of internet technologies is still in the ascendant, internet has being used widely in the world. Intranet, by using the key technologies of internet such as TCP/IP，HTTP，WWW，also has get a rapid development in the enterprise information system application. However，most of the enterprise intranet system in use faces many problems. For example, administrator has no idea to manage a variety of information available in a unified intranet system，include non-structured and semistructured data such as documents and structured database resources. Moreover, there are many faults in access control to secret information.

Metadata provides a abstract model for managing information. by Using Object-Oriented technologies and the characteristics of RDBMS，metadata describes a kind of type information as a object which is composed of based attributes and extended attributes. As a result，administrator can control the user's access to each extended attributes of a object.

Object-Oriented approach will substantially enhance collaboration over diverse information resources. Access control to field of records becomes feasible and easy because we can control easily a object and its attributes. We believe access control model based on metadata represents a kind of new access control approach of intranet system in the future.

Keywords：enterprise network, metadata, access control, Object-Oriented technology, software model
