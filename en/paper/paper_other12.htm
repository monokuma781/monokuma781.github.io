 用户级通信在软件分布式共享存储系统中的应用

 
用户级通信在软件分布式共享存储系统中的应用


本文在不改变软件分布式共享存储系统协议的前提下对系统性能提高的途径和具体策略进行了研究，发现用户级通信可以明显地降低通信延迟、提高通信带宽，并尽可能地减少软件开销和数据拷贝次数，为软件分布式共享存储系统性能的提高开辟了新的道路。

首先，本文对共享存储和分布式共享存储研究的现状进行了综述，并详细介绍了软件分布式共享存储和用户级通信的各项关键技术。

在对基于锁的高速缓存一致性协议进行研究的基础上，我们提出了一个独特的软件分布式共享存储系统的并行效率评价模型，其独特之处在于它引入了对软件开销的考虑，并且使用消息延迟参数而不是带宽来对软件分布式共享存储系统的通信时间下限进行估计。对并行效率评价模型的分析，表明用户级通信技术的确可以进一步改善通信底层的性能，从而使系统整体性能得到优化。

在对软件分布式共享存储系统的通信行为进行系统分析的基础上，我们总结出其通信底层区别于一般用户级通信库的性质和特点，并基于这样的特点设计和实现了一个完整而独特的用户级通信库函数JMCL。

实验结果表明，JMCL极大地改善了JIAJIA系统的通信性能，并因此而使得其上的应用程序的性能获得了明显的改善，尤其是那些过去由于通信密集而被认为比较不适合于软件分布式共享存储系统的应用程序，它拓展了JIAJIA系统的适用范围，可扩展性更好。

通过对JMCL及其上层的软件分布式共享存储系统JIAJIA的性能测试和分析，我们总结出用户级通信库影响软件分布式共享存储系统性能的本质特征：它不仅改善了系统的通信性能，同时它也使得系统的并行计算性能得到明显改善，从而十分显著地提高了软件分布式共享存储系统的整体性能。

关键词 软件分布式共享存储，用户级通信，集群。

User-level Networking for Software DSMs
Yongjie Mao (Computer Architecture)
Directed by Prof. Mingfa Zhu

In this dissertation we study the ways and practical policies to improve the performance of software distributed shared memories(DSMs) without any changes in the protocols.

Firstly we provide an overview of research on shared memory and distributed shared memory. Then we present the key issues about software DSMs and user-level networking in details.

After studying the lock-based cache coherence protocol we present a new efficiency model for software DSMs. Its new features include adding software overhead to computing and using latency instead of bandwidth to obtain the lower limit of communication time.

Based on the analysis of this new efficiency model we find that the performance of software DSMs can be improved indeed by adopting the user-level networking. Moreover, the unusual behavior of factor δ in the model shows a exciting way to improve the performance of software DSMs further.

We study the communication behavior of a software DSM system called JIAJIA thoroughly and summarize its characteristics that are very different with those of the normal user level communication libraries. Then we design and implement a complete user level communication library for JIAJIA called JMCL.

With the analysis of the performance of JMCL and JIAJIA system, we draw a conclusion about the effects of user-level networking on software DSMs: User-level networking not only improves the performance of the communication substrate and reduces the software overhead but also improves the parallel computing performance so that it improves the entire performance of software DSMs significantly.

Keywords: software DSMs, user-level networking, clusters.
