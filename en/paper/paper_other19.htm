机群管理系统的结构及其实现


异构机群系统是指将一组异构结点计算机通过高性能互连网络连接起来，并作为一个单一系统向其用户提供高性能、高可用服务的系统。在每个结点计算机上安装有不同商业化的操作系统（AIX、LINUX、NT等），整个异构机群建立在这样一些独立结点的连接之上，可见管理软件在异构机群系统的重要地位。本文是作者对曙光3000异构机群管理系统研究开发工作的总结和深入。论文首先简要介绍了机群和异构机群的概念和异构机群管理所覆盖的内容；然后对目前机群管理系统可以采用的体系结构、通讯协议、标准和一些新技术作了分析和比较，探讨了它们各自的优劣及今后的发展趋势；紧接着又以曙光3000异构机群管理系统为实例，介绍了它的技术路线，和一些关键问题的解决思想和方案，并且对曙光3000异构机群管理系统主要模块的系统结构模型、以及实现性能作了详细的论述；最后，论文向读者展示了曙光3000异构机群管理系统的重要功能和部分界面，并提出该系统还存在的问题和今后的改进方向。

关键词： 异构机群 异构机群管理系统 代理 RMI SNMP CORBA JAVA 虚拟命令层

Heterogenous Cluster System is a system which is connected with a group of node computers based on Heterogenous commercial operation system via high performance network and provides high performance, high availability services to its users as a single system. Heterogenous commercial operation system installed on every node computer, the whole cluster is constructed on the connection between those independent nodes. Thus, we can see the important role of management software in heterogenous cluster systems. This paper is summarized from the research and development work of Dawning3000 Heterogenous Cluster System Management System. Firstly, the concept of cluster and the areas covered by Cluster System Management System (CSMS) are briefly introduced; then, the author analyses and compares the communication protocols, architectures, standards and some new techniques which can be used by Heterogenous Cluster System Management Systems now, and discusses their advantages and shortcomings to show the trend of future; after this, the author expounds the technical routes and the key solutions of Dawning3000, and discusses the system model and the research of the Heterogenous Cluster System Management System's scalability in great detail; at last, the author shows readers the main functions and some interfaces, and proposes the problems needed to be improved.

KEYWORDS: Heterogenous Cluster Cluster System Management System Agent RMI SNMP CORBA Virtual Command Layer
